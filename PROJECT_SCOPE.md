# Scope Included

## Three Faces
1. Customer-facing public site
2. Client dashboard
3. Employee/ops/admin + CRM

## Bells and Whistles Included
- Voice AI endpoints
- Twilio calling webhook
- SMS webhook
- OpenAI chat wrapper
- Optional client OpenAI keys
- Optional platform OpenAI key
- Optional Twilio credentials
- Fallback replies if no key exists
- Client-specific AI instances
- Embeddable iframe widget
- Industry templates
- Unsupported industry consultation request
- Internal sales CRM
- Client-customer lead CRM
- Conversation logging
- Call/SMS logging
- Django admin CRUD for everything
